CatMash
---

The app is deployed on Heroku: [Click here](https://kk-cat-mash.herokuapp.com)

## 1.Compile
### Maven
Run command :
```
mvn install
```

## 2.Execute
### Java
The compile command build an executable fat jar.
Run the jar with the command :
```
java -jar catmash.jar
```
